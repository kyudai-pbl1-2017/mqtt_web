MQTT Web連携サンプル（WebSocket利用）
=====================================


システム要件
============

* HTTPサーバ

設定
====

``js/mqtt_client.js`` 内の ``mqtt_host`` 及び ``topic_base`` にそれぞれMQTTサーバ情報とトピック名のベースパスを設定する。
